# Windows Server 보안 취약점 진단 스크립트

## 1. 개요

이 리포지토리는 KISA(한국인터넷진흥원)의 '주요정보통신기반시설 기술적 취약점 분석·평가 상세 가이드'를 기반으로, Windows Server의 보안 취약점을 자동으로 진단하고 조치하는 PowerShell 스크립트 모음입니다.

## 2. 프로젝트 구조

각 `W-XX` 디렉터리는 KISA 보안 가이드의 특정 취약점 항목에 해당합니다. 각 디렉터리 내에는 다음과 같은 파일들이 포함되어 있습니다.

- **메인 스크립트 (`W-XX.ps1`)**: 해당 취약점을 점검하고, 사용자 확인 후 자동으로 조치하는 메인 스크립트입니다.
- **취약점 재현 스크립트 (`set_vulnerabilities.ps1`)**: 테스트 목적으로 시스템을 의도적으로 취약한 상태로 만드는 스크립트입니다.
- **상세 문서 (`*.md`)**: 취약점의 원리, 스크립트의 동작 방식, 사용법 등을 상세히 설명하는 문서들입니다.

## 3. 현재 제공되는 스크립트

- **`W-24`**: NetBIOS 바인딩 서비스 구동 점검
- **`W-63`**: DNS 서비스 동적 업데이트 설정 점검
- **`W-78`**: 보안 채널 데이터 디지털 암호화/서명 점검

## 4. 사용 방법

1.  점검을 원하는 항목의 `W-XX` 디렉터리로 이동합니다.
2.  PowerShell을 **관리자 권한**으로 실행합니다.
3.  메인 스크립트(`W-XX.ps1`)를 실행합니다.
4.  상세한 사용법은 각 디렉터리 내의 `README.md` 파일을 참고하십시오.

## 5. 주의사항

- 본 스크립트는 시스템의 레지스트리 및 보안 설정을 변경할 수 있습니다.
- 스크립트의 내용을 충분히 숙지하신 후 사용하시기 바랍니다.
- 운영 환경에 적용하기 전에 반드시 테스트 환경에서 충분한 검증을 거치는 것을 권장합니다.
